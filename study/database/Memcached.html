数据操作 #

1.添加数据

add  key   0|1   失效时间   数据长度

参数说明： key :键的名称 0|1：是否压缩，0表示不压缩，1表示要压缩， 失效时间：缓存周期，单位是秒 数据长度：单位是字节

tip:如果键已经存在则添加失败

2.查询数据

get name

3.修改数据

replace   key   0|1   失效时间   数据长度

tip:如果键不存在则修改失败

4.设置数据

set key   0|1   失效时间   数据长度

tip:如果键已经存在，则是修改，如果键没有存在，则是添加

5.删除数据

删除单个数据

delete key

删除全部数据

flush_all

6.加减指令

incr  key   数值
decr  key   数值
